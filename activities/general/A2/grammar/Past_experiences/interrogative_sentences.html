Interrogative sentences

How to form questions with simple past?

To form Interrogative sentences in simple past you also need the help of the auxiliary verb did.

Structure: (Wh- word) Did + subject + infinitive verb + rest of the sentence + question mark

There are two types of questions: yes/ no questions and information questions, here you can see how they can be formed.

Remember that when you use the auxliary verb did  you do not need to modify the main verb.

If the question starts with DID, you can give a short answer using DID. It is not necessary to use the main verb in the answer.

Examples

Did you sleep well last night?
Yes, I did / No, I didn't.

Did the airplane arrive on time?
Yes, it did / No, it didn't.

Let's practice what you learned!

Analyze each question, choose true if it is written correctly, if not choose false. 

Example:  When did you arrive home?   - true

Question 1

1. What did you eat last Monday?

Feedback

True

It is correct!

Question 2

2. What time the train left?

Feedback

False

The auxiliary verb is missing. 

* What time did the train leave?

Question 3

3. Did he took my book yesterday? Yes, he did.

Feedback

False

The question is incorrect. The main verb does not change when we have the auxiliary verb did.

* Did he take my book yesterday? Yes, he did. 

Question 4

4. How much did this dress cost? 

Feedback

True

It is correct! Cost is an irregular verb that does not change in simple past.

Question 5

5. Where did Jane goed on holiday last month?

Feedback

False

The question is wrong. The verb go is conjugated incorrectly, goed does not exist. 

* Where did Jane go on holiday last month?

Question 6

6. Did you watch the movie last night?

Feedback

True

It is correct!

Question 7

7. Did it rained a lot on your holiday?

Feedback

False

This is the correct form: Did it rain a lot on your holiday?

Question 8

8. Who did you visit last winter?

Feedback

True

It is correct like that!
