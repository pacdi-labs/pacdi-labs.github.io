Bathroom

The bathroom

Now, we are in my bathroom!

Can you identify some of the objects in the picture?

  • There are some plants.
  • There is one toilet.
  • There is a mirror. 
  • There are two white towels.

Etc, etc.

There are many objects in my bathroom, but there isn't a chair.

There isn't a toothpaste. 

There aren't any toothbrushes in my bathroom.

I can mention the things that exist in my bathroom, and mention the things that DON'T.

Negative Form

The negative is formed by putting not after is or are:

  • There is not a soap in the shower.
  • There are not clean towels.

Contractions

  • There is not = There isn't
  • There are not = There aren't

There aren't with ANY

When we want to indicate that a ZERO QUANTITY of something we use there aren't any.

  • There aren't any rolls of toilet paper.
  • There isn't any shampoo in the shower.
