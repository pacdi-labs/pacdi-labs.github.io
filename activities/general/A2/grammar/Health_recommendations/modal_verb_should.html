Modal verb "should"

Read the information, if it is necessary take notes in your portfolio. After that, answer the activities with the information given.

When do we use this verb?

We use should when we want to show what is right, appropriate, etc., especially when criticizing somebody’s actions.

  • You shouldn't drink and drive.                              
  • He should be more careful.

We use should when we want to give or ask for advice (recommendation).

  • You should stop worrying about it.
  • Should I call him and apologize?

Note that:

  • The auxiliary verb should is invariable. There is only one form: should 

    Be careful:  It is imposible to say shoulds

We don't say: She shoulds go to school.                            We say: She should go to school. 

  • The main verb is usually in the base form (He should go).

Be careful: we do not add  to  before the main verb. 

We don't say: You should to go to school.                                       We say: You should go to school.     

  • There is no short form for should, but we can shorten the negative should not to shouldn't.

We can say: I should not eat many candies.                 We can also say: I shouldn't eat many candies.
