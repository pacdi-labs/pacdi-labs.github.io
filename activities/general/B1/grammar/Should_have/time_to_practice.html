Time to practice

It's your turn!

Easy, isn't it?

 Now, it's time for you to practice. Write two or three sentences about each situation (the things that you should or shouldn’t have done). For example:

  • I shouldn't have said those horrible things when I was angry.

  •  Check the grammar structure of should have + participle.
  •  Check that you used the correct conjugation of the verbs in past participle.
  •  If you don't know how to say a word, use the online dictionary.

 Did you like this activity?

  • Write a short personal reflection about this activity. These questions might help you:
    • Did I learn something new today?
    • Was it difficult? Was it easy?
    • Is this a useful structure to improve my skills?
    • What do I need to learn more?
  • Create a glossary with all the new words you learned in this activity.
